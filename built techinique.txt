1) BUILD TECHNIQUE (STACK + ARCHITECTURE)
================================================================
Frontend
- React 19 + TypeScript (Vite 6) with component-driven UI.
- Tailwind CSS (via @tailwindcss/vite) for styling.
- Motion (motion/react) for transitions and UI animation.
- Lucide icons for all button glyphs.

Media + Graphics
- HTML Canvas for drawing regions, previews, and diff overlays.
- Web Workers for heavy media tasks (video export, overlay export, progress bar export, frame extraction, super image export).
- WebCodecs for encoding (H.264/MP4 and AV1/WebM depending on export choice).
- Audio mixing + decoding (custom utilities + asset stores).
- JSZip for bulk image exports (zip download).

AI + Automation
- Serverless API: /api/detect-differences uses Google Gemini (model: gemini-3.1-pro-preview) for difference detection.
- Client-side auto detection for fast diff detection (image processing + morphological operations + alignment checks).
- Optional AI upscaling (ESRGAN models + MAXIM deblur) via UpscalerJS + TensorFlow.js.

Storage & Persistence
- Local settings and presets (localStorage-based services).
- User package library, background packs, watermark presets, timestamp presets, saved layout snapshots.
- File System Access API (when available) to export Super Image batches directly to a chosen folder.

App Structure
- Single-page app: App.tsx controls global mode state and persistent sub-modes.
- Modes are component-based: Upload, Editor, Play, Video Setup/Preview, Overlay Editor, Splitter, Upscaler, Backgrounds, Timers, Progress Bars, Frame Extractor, Watermark Removal.
- Navigation is state-driven and persisted to history via window.history.pushState for back/forward behavior.

2) CORE DATA MODEL (WHAT THE APP STORES/USES)
================================================================
Puzzle
- imageA: base image (data URL / URL)
- imageB: modified image (data URL / URL)
- regions: list of normalized rectangles (x, y, width, height)
- title (optional)

PuzzleSet
- title, version, puzzles[]

VideoSettings (shared across Video Mode, Timer Mode, Progress Bar Mode)
- Aspect ratio, style stack, reveal behavior, durations, text templates, audio settings, export settings, logo settings, generated backgrounds, custom layout settings.

Generated Background Packs
- Named packs with 100 recipe entries, each recipe includes motif family, palette, seed, density, safe zone, motion speed, etc.

Video Package Library
- Named packages: each stores shared settings + per-aspect layout snapshots.

4) ALL APP MODES (HIGH-LEVEL OVERVIEW)

Progress Bar Mode

- Draw watermark regions on both images, swap pixel areas, save presets, export cleaned images.

Incomplete Pairs Modal

Detection Method Modal
- Manual Selection: skips auto detection and opens manual Editor.

- Cancel: closes modal and clears pending split queue.

Review Puzzles Modal
- Edit Puzzle (pencil): opens Editor Canvas for that puzzle.
- Remove Puzzle (trash): removes from review list.
- Cancel: closes review without submitting.
- EXPORT JSON: downloads PuzzleSet JSON.
- EXPORT IMAGES: downloads paired images as separate files.
- EXPORT VIDEO: routes batch to Video Setup export flow.
- START GAME: confirms batch and starts Play mode.

Edit Puzzle Modal
- Close (X): closes editor and returns to review list.

Other Functional Behavior
- Drag-and-drop supports batch with names like name.png + namediff.png.
- Missing pairs are detected and fixed via modal.
- Processing overlays show progress and status messages.

21) CORE FUNCTIONAL PIPELINES (WHAT EACH MAJOR FUNCTION DOES)
================================================================
Difference Detection
- Manual: user draws boxes in Editor.
- Auto (client): computes image diff on canvas, handles alignment and morphology to detect regions.
- AI: /api/detect-differences uses Gemini to output normalized bounding boxes.

Batch Management
- Upload mode can collect multiple puzzles into a batch.
- Review modal allows editing/removal before starting game or export.

Video Export
- VideoSettingsPanel sets styles, text, timing, audio, layout, reveal behavior.
- VideoPlayer preview uses same settings.
- Export uses WebCodecs in worker, supports H.264 and AV1.

Overlay Export
- OverlayVideoEditor merges base media + overlays + batch images + audio.
- Export builds timelines for each output and encodes via worker + WebCodecs.

Progress Bar Export
- ProgressBarMode renders animated bar or text-fill countdown per theme.
- Export pipeline uses worker + WebCodecs.

Frame Extraction
- FrameExtractor uses shared timestamps for multiple clips.
- Optional super aligner builds split pairs from those frames.

Watermark Removal
- Swaps identical pixel rectangles between puzzle and diff images.
- Presets allow repeatable selections.

Upscaling
- Fast Enhance: heuristic post-processing (detail boost, contrast, edge threshold).
- AI Super-Resolution: ESRGAN + optional MAXIM deblur.

Background Generation
- Procedural motifs + palettes stored as packs, seeded for deterministic output.
